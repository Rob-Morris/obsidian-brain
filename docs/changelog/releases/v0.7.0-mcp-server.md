# MCP Server (v0.3.0–v0.7.0)

Shipped: 2026-03-16

## Overview

The brain becomes programmable.

## Highlights

- **Brain MCP server**: 3 tools (`brain_read`, `brain_search`, `brain_action`), stdio transport, Python ≥3.10 + `mcp` SDK, startup auto-compile/build, and direct script imports rather than shelling out to `main()`.
- **Artefact type and routing foundations**: the first artefact library plus `compile_router.py` as the compiled-router foundation.
- **Retrieval stack**: optional Obsidian CLI integration plus BM25 indexing/search with typed filtering and snippets.
- Early test coverage and startup robustness for the first programmable brain runtime.
