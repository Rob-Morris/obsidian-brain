# Infrastructure (v0.13.0–v0.16.0)

Shipped: 2026-03-28

## Overview

Restructured machine-local state and established the first proper migration layer.

## Highlights

- **`.brain/` directory**: machine-local caches moved from `_Config/` dotfiles to `.brain/local/` (gitignored). New vault-portable files: `.brain/preferences.json`, `.brain/tracking.json`.
- **Migration framework**: `scripts/migrations/` with versioned scripts that run automatically during upgrade, making structural changes explicit and repeatable.
- **Naming and mutation foundations**: generous filenames, the `~` temporal separator, tool privilege split, section targeting, basename resolution, broken-link auto-repair, and artefact library manifests/schemas.
- **Bootstrap and operator setup**: `init.py`, core skills, compliance tooling, colour-system refinements, and user-facing documentation.
