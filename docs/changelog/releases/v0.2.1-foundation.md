# Foundation (v0.1.0–v0.2.1)

Shipped: 2026-03-15

## Overview

The project establishes its first versioned core path, repo conventions, and release discipline.

## Highlights

- `.brain-core/v0.2.x/` becomes the versioned install path, with repo-wide reference rewrites and the template-vault symlink aligned to the new layout.
- `Agents.md` with git conventions, `CLAUDE.md` symlink, `VERSION` as the semver source of truth, and changelog maintenance discipline.
- Version numbering rebased to start at `v0`, making the whole system explicitly pre-1.0 and semver-driven.
