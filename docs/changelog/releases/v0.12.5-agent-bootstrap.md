# Agent Bootstrap (v0.8.0–v0.12.5)

Shipped: 2026-03-26

## Overview

The first usable agent handshake lands: session bootstrap, upgrade support, workspaces, and a broader artefact surface.

## Highlights

- **`brain_session`** (v0.12.0): top-level MCP tool for agent bootstrap. Returns compiled, token-efficient payload in one round trip: always-rules, user preferences, gotchas, triggers, condensed artefact types, environment, memory/skill/plugin/style indexes.
- **Upgrade and workspace foundations**: `brain_action("upgrade")`, standalone `upgrade.py`, workspace slug-to-path resolution, and register/unregister actions.
- **Broader artefact surface**: People, Observations, Cookies, Captures, Workspaces, Presentations, `brain_read(resource="file")`, and stronger BM25 title boosting.
- **Server/runtime polish**: tool privilege split, module reload on version drift, auto-recompile on taxonomy changes, and `_Assets/` restructuring.
