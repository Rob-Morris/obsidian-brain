- Preserve existing structural resolver semantics for headings with child headings, heading body and section ranges, and `next_boundary_raw` reporting.
- Add resolver-level and MCP-level regression coverage in `tests/test_markdown_structural.py` and `tests/test_mcp_server.py` for `brain_edit(operation="edit", target="## Alpha", scope="intro")` preserving following sibling sections.
- Move `cli/brain` `BRAIN_INSTALL_REF` to `v0.49.2`; `BRAIN_CLI_VERSION` is unchanged because the CLI dispatch surface did not change.
