# v0.42.2 — Add bootstrap-only folder binding to init.py

- Add `init.py --skip-mcp` as an explicit bootstrap-only mode for folder targets, so Brain can scaffold workspace/binding state without provisioning the managed runtime or writing MCP/client config.
- Scaffold `.brain/local/workspace.yaml` through that bootstrap-only path and keep it narrow: no `.mcp.json`, `.codex/config.toml`, `CLAUDE.md`, or SessionStart hook writes when `--skip-mcp` is used.
- Add Brain-owned machine-local ignore management for git-backed target folders, updating tracked `.gitignore` when one exists and falling back to `.git/info/exclude` only when no `.gitignore` is present.
- Document the new binding mode across the user, script, and architecture references, and pin it with focused `tests/test_init.py` coverage.
