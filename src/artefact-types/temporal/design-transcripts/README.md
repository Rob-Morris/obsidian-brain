# Design Transcripts

Q&A refinement transcripts tied to a source artefact. Each transcript is bound to a single source document.

## Install

```
_Config/Taxonomy/Temporal/design-transcripts.md        ← taxonomy.md
_Config/Templates/Temporal/Design Transcripts.md       ← template.md
_Temporal/Design Transcripts/                          ← create folder
```

### Router trigger (optional)

```
- After Q&A refinement → [[_Config/Taxonomy/Temporal/design-transcripts]]
```
