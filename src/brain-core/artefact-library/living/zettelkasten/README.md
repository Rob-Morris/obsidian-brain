# Zettelkasten

Auto-maintained atomic concept mesh. One card per concept, dense links to sources and related ideas.

## Style

Suggested colour: mint (`--palette-mint`, `#A8E8D0`). Use `style.css` as-is or adapt to your vault's colour scheme.

## Install

```
_Config/Taxonomy/Living/zettelkasten.md  ← taxonomy.md
_Config/Templates/Living/Zettelkasten.md ← template.md
Zettelkasten/                            ← create folder
style.css                                ← merge into .obsidian/snippets/folder-colours.css
```
