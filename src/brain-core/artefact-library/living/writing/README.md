# Writing

Long-form written works: essays, articles, stories, books. Lifecycle from draft through to published.

## Style

Suggested colour: rose (`--palette-rose`, `#F2A8C4`). Use `style.css` as-is or adapt to your vault's colour scheme.

## Install

```
_Config/Taxonomy/Living/writing.md          ← taxonomy.md
_Config/Templates/Living/Writing.md         ← template.md
Writing/                                    ← create folder
style.css                                   ← merge into .obsidian/snippets/folder-colours.css
```
