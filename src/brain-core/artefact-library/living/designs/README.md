# Designs

Design documents, wireframes, mockups, and design-related notes. One file per design.

## Style

Suggested colour: coral (`--palette-coral`, `#F0908F`). Use `style.css` as-is or adapt to your vault's colour scheme.

## Install

```
_Config/Taxonomy/Living/designs.md       ← taxonomy.md
_Config/Templates/Living/Designs.md      ← template.md
Designs/                                 ← create folder
style.css                                ← merge into .obsidian/snippets/folder-colours.css
```
