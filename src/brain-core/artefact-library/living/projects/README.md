# Projects

Project index files. One file per project, linking to all related artefacts via the project tag.

## Style

Suggested colour: sage (`--palette-sage`, `#8FCA8E`). Use `style.css` as-is or adapt to your vault's colour scheme.

## Install

```
_Config/Taxonomy/Living/projects.md      ← taxonomy.md
_Config/Templates/Living/Projects.md     ← template.md
Projects/                                ← create folder
style.css                                ← merge into .obsidian/snippets/folder-colours.css
```
