# Wiki

Interconnected knowledge base. One page per concept, linked liberally.

Included in the template vault.

## Style

Suggested colour: mint (`--palette-mint`, `#A8E8D0`). Use `style.css` as-is or adapt to your vault's colour scheme.

## Install

```
_Config/Taxonomy/Living/wiki.md          ← taxonomy.md
_Config/Templates/Living/Wiki.md         ← template.md
Wiki/                                    ← create folder
style.css                                ← merge into .obsidian/snippets/folder-colours.css
```
