# Documentation

Guides, standards, and reference material. Evolves over time as understanding deepens.

## Style

Suggested colour: peach (`--palette-peach`, `#F5B88A`). Use `style.css` as-is or adapt to your vault's colour scheme.

## Install

```
_Config/Taxonomy/Living/documentation.md       ← taxonomy.md
_Config/Templates/Living/Documentation.md      ← template.md
Documentation/                                 ← create folder
style.css                                      ← merge into .obsidian/snippets/folder-colours.css
```
