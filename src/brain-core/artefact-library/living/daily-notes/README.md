# Daily Notes

High-level end-of-day summaries. One file per day, distilled from the day's log.

## Style

Suggested colour: sky (`--palette-sky`, `#8BB8E8`). Use `style.css` as-is or adapt to your vault's colour scheme.

## Install

```
_Config/Taxonomy/Living/daily-notes.md     ← taxonomy.md
_Config/Templates/Living/Daily Notes.md    ← template.md
Daily Notes/                               ← create folder
style.css                                  ← merge into .obsidian/snippets/folder-colours.css
```
