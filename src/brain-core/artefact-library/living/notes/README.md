# Notes

Flat knowledge base of interconnected notes. One page per concept, date-prefixed.

## Style

Suggested colour: teal (`--palette-teal`, `#7DD6D2`). Use `style.css` as-is or adapt to your vault's colour scheme.

## Install

```
_Config/Taxonomy/Living/notes.md         ← taxonomy.md
_Config/Templates/Living/Notes.md        ← template.md
Notes/                                   ← create folder
style.css                                ← merge into .obsidian/snippets/folder-colours.css
```
