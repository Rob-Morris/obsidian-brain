# Research

In-depth research notes and findings on specific topics. One topic per file, date-prefixed.

## Style

Suggested colour: teal → rose blend (`#A5C5CD`). See `[[.brain-core/colours]]` for the temporal blend formula. Use `style.css` as-is or adapt to your vault's colour scheme.

## Install

```
_Config/Taxonomy/Temporal/research.md          ← taxonomy.md
_Config/Templates/Temporal/Research.md         ← template.md
_Temporal/Research/                            ← create folder
style.css                                      ← merge into .obsidian/snippets/folder-colours.css
```
