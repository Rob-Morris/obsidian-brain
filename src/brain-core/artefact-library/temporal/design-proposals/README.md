# Design Proposals

Structured recommendations for contemplated changes that need a decision before action.

## Install

```
_Config/Taxonomy/Temporal/design-proposals.md   <- taxonomy.md
_Config/Templates/Temporal/Design Proposals.md  <- template.md
_Temporal/Design Proposals/                     <- create folder
```

### Router trigger (optional)

```
- When vault work surfaces a contemplated design change → [[_Config/Taxonomy/Temporal/design-proposals]]
```
