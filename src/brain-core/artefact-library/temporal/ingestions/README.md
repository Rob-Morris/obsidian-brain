# Ingestions

Processing records for content decomposition — how content was analysed, broken down, and absorbed into the vault.

## Install

```
_Config/Taxonomy/Temporal/ingestions.md     <- taxonomy.md
_Config/Templates/Temporal/Ingestions.md    <- template.md
_Temporal/Ingestions/                       <- create folder
```

### Router trigger (optional)

```
- When ingesting content that warrants decomposition → [[_Config/Taxonomy/Temporal/ingestions]]
```
