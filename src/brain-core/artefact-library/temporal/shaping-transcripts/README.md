# Shaping Transcripts

Q&A refinement transcripts tied to a source artefact. Each transcript is bound to a single source document. Use these to capture the back-and-forth process of shaping any artefact through questions and answers.

## Style

Suggested colour: lavender → rose blend (`#CCA9DB`). See `[[.brain-core/colours]]` for the temporal blend formula. Use `style.css` as-is or adapt to your vault's colour scheme.

## Install

```
_Config/Taxonomy/Temporal/shaping-transcripts.md        ← taxonomy.md
_Config/Templates/Temporal/Shaping Transcripts.md       ← template.md
_Temporal/Shaping Transcripts/                          ← create folder
style.css                                               ← merge into .obsidian/snippets/folder-colours.css
```

### Router trigger (optional)

```
- After Q&A refinement → [[_Config/Taxonomy/Temporal/shaping-transcripts]]
```
