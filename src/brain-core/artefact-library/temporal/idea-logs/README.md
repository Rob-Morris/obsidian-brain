# Idea Logs

Quick captures of new ideas in rough form. Low-friction, date-prefixed, tagged entries.

## Common progressions

These are patterns, not a required pipeline:

1. **Idea log** — raw capture, a few sentences
2. **Living artefact** — fleshed out, explored
3. **Design** — shaped proposal with decisions and structure

## Install

```
_Config/Taxonomy/Temporal/idea-logs.md   ← taxonomy.md
_Config/Templates/Temporal/Idea Logs.md  ← template.md
_Temporal/Idea Logs/                     ← create folder
```

### Router trigger (optional)

```
- When a new idea strikes → [[_Config/Taxonomy/Temporal/idea-logs]]
```
