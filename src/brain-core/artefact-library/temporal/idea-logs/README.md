# Idea Logs

Quick captures of new ideas in rough form. Low-friction, date-prefixed, tagged entries that can graduate into living artefacts and eventually designs.

## Graduation path

1. **Idea log** — raw capture, a few sentences
2. **Living artefact** — fleshed out, explored
3. **Design** — shaped proposal with decisions and structure

## Style

Suggested colour: blush → rose blend (`#ECB2B7`). See `[[.brain-core/colours]]` for the temporal blend formula. Use `style.css` as-is or adapt to your vault's colour scheme.

## Install

```
_Config/Taxonomy/Temporal/idea-logs.md   ← taxonomy.md
_Config/Templates/Temporal/Idea Logs.md  ← template.md
_Temporal/Idea Logs/                     ← create folder
style.css                                ← merge into .obsidian/snippets/folder-colours.css
```
