# Presentations

Slide decks generated from markdown content using Marp CLI. Source is markdown; output is PDF in `_Assets/Generated/Presentations/`.

## Install

```
_Config/Taxonomy/Temporal/presentations.md      <- taxonomy.md
_Config/Templates/Temporal/Presentations.md     <- template.md
_Config/Skills/presentations/SKILL.md           <- SKILL.md
_Config/Skills/presentations/theme.css          <- theme.css
_Temporal/Presentations/                        <- create folder
```

### Router trigger (optional)

```
- When creating a presentation or slide deck → [[_Config/Taxonomy/Temporal/presentations]]
```
