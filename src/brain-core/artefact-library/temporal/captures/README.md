# Captures

External material ingested verbatim — emails, meeting notes, Slack threads, data extracts. Frozen on ingest, never edited.

## Install

```
_Config/Taxonomy/Temporal/captures.md      <- taxonomy.md
_Config/Templates/Temporal/Captures.md     <- template.md
_Temporal/Captures/                        <- create folder
```

### Router trigger (optional)

```
- When ingesting external material → [[_Config/Taxonomy/Temporal/captures]]
```
