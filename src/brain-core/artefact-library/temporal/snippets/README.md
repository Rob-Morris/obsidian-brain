# Snippets

Short, crafted content pieces derived from existing work — X posts, blurbs, descriptions, taglines. Each snippet is a point-in-time capture linked to its source via provenance.

## Style

Suggested colour: gold → rose blend (`#EBC49E`). See `[[.brain-core/colours]]` for the temporal blend formula. Use `style.css` as-is or adapt to your vault's colour scheme.

## Install

```
_Config/Taxonomy/Temporal/snippets.md       ← taxonomy.md
_Config/Templates/Temporal/Snippets.md      ← template.md
_Temporal/Snippets/                         ← create folder
style.css                                   ← merge into .obsidian/snippets/folder-colours.css
```

### Router trigger (optional)

```
- When crafting a shareable piece of text → [[_Config/Taxonomy/Temporal/snippets]]
```
