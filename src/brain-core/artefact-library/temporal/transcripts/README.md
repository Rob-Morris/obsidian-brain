# Transcripts

Conversation transcripts. One file per conversation, preserving the exchange as it happened.

Included in the template vault.

## Style

Suggested colour: mauve → rose blend (`#D8A4C2`). See `[[.brain-core/colours]]` for the temporal blend formula. Use `style.css` as-is or adapt to your vault's colour scheme.

## Install

```
_Config/Taxonomy/Temporal/transcripts.md       ← taxonomy.md
_Config/Templates/Temporal/Transcripts.md      ← template.md
_Temporal/Transcripts/                         ← create folder
style.css                                      ← merge into .obsidian/snippets/folder-colours.css
```

### Router trigger (optional)

```
- After conversation worth preserving → [[_Config/Taxonomy/Temporal/transcripts]]
```
