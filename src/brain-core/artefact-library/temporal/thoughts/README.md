# Thoughts

Raw, unformed thinking captured in the moment. Low bar for capture — most won't go anywhere, and that's fine.

## Install

```
_Config/Taxonomy/Temporal/thoughts.md       ← taxonomy.md
_Config/Templates/Temporal/Thoughts.md      ← template.md
_Temporal/Thoughts/                         ← create folder
```

### Router trigger (optional)

```
- When a raw thought surfaces → [[_Config/Taxonomy/Temporal/thoughts]]
```
