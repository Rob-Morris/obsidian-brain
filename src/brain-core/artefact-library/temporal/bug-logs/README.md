# Bug Logs

Point-in-time records of broken behaviour — something that is objectively wrong or producing incorrect output.

## Install

```
_Config/Taxonomy/Temporal/bug-logs.md   <- taxonomy.md
_Config/Templates/Temporal/Bug Logs.md  <- template.md
_Temporal/Bug Logs/                     <- create folder
```

### Router trigger (optional)

```
- When encountering something that is objectively broken or producing wrong output -> [[_Config/Taxonomy/Temporal/bug-logs]]
```
