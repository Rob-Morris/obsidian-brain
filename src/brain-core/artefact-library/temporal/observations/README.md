# Observations

Timestamped facts, impressions, and things noticed. Low bar for capture — if you learned something worth remembering, write it down.

## Install

```
_Config/Taxonomy/Temporal/observations.md       ← taxonomy.md
_Config/Templates/Temporal/Observations.md      ← template.md
_Temporal/Observations/                         ← create folder
```

### Router trigger (optional)

```
- When you notice or learn a discrete fact worth recording → [[_Config/Taxonomy/Temporal/observations]]
```
