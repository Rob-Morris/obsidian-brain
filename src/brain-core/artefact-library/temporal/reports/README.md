# Reports

Overviews of detailed processes — research, diagnosis, investigation, analysis, audit. Distils findings and implications.

## Install

```
_Config/Taxonomy/Temporal/reports.md        ← taxonomy.md
_Config/Templates/Temporal/Reports.md       ← template.md
_Temporal/Reports/                          ← create folder
```

### Router trigger (optional)

```
- After completing a detailed process → [[_Config/Taxonomy/Temporal/reports]]
```
