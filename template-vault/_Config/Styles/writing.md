# Writing Style

## Universal

Use Australian English.

## External Audience

**Applies when:** the file is tagged `audience/external`, or the user explicitly asks for polished prose.

1. Put the point first. Support it underneath.
2. Vary sentence length: short sentences punch, long ones build momentum, the mix keeps prose alive.
3. Use short, familiar, specific words. "Use" not "utilise"; "pickup truck" not "vehicle."
4. Choose strong verbs that carry their own weight. Cut the adverb.
5. No em dashes. Use commas, colons, and semi-colons.
6. Write how a sharp person talks. If you wouldn't say it aloud, rewrite it.
7. Avoid inflated vocabulary and filler phrases. If a simpler word exists, use it.
8. Show, don't tell. Concrete detail persuades; abstract claims don't.
9. Every sentence earns its place. Stop when you're done; never summarise what you just said.
10. Lead each sentence with the important thing. Push setup and qualifiers to the end.
